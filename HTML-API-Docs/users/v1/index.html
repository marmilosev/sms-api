# Here is a curl example for GET user
curl --location 'http://localhost:8080/v1/users'

# Here is a curl example for POST user
curl --location 'http://localhost:8080/v1/users/add' \
--header 'Content-Type: application/json' \

# Here is a curl example for PUT user
curl --location --request PUT 'http://localhost:8080/v1/users/[id]' \
--header 'Content-Type: application/json' \

# Here is a curl example for PUT user
curl --location --request DELETE 'http://localhost:8080/v1/users/[id]'

DESCRIPTION FOR ERROR

Code Message URL Description
1 Error with first name input https://mmilosevic-diplomski-api.com/users/v1/1 This error occurs when you don't provide any data for user's first name.
2 Error with last name input https://mmilosevic-diplomski-api.com/users/v1/2 This error occurs when you don't provide any data for user's last name.
3 Error with username input https://mmilosevic-diplomski-api.com/users/v1/3 This error occurs when you don't provide any data for user's username.
4 Error with number input https://mmilosevic-diplomski-api.com/users/v1/4 This error occurs when you don't provide any data for user's number.
5 Error with mail input https://mmilosevic-diplomski-api.com/users/v1/5 This error occurs when you don't provide any data for user's mail.
6 Error with mail input - Mail has to be in standard format for mail https://mmilosevic-diplomski-api.com/users/v1/6 This error occurs when you provide email without '@' sign for user's mail.
7 Error with password input https://mmilosevic-diplomski-api.com/users/v1/7 This error occurs when you don't provide any data for user's password.
8 User successfully created/inserted in database. https://mmilosevic-diplomski-api.com/users/v1/8 This is NOT error, just confirmation of success. Data for user are inserted in database.
9 User successfully updated. https://mmilosevic-diplomski-api.com/users/v1/9 This is NOT error, just confirmation of success. Certain data for user with certain id is successfully updated.
10 User successfully registered and inserted in database. https://mmilosevic-diplomski-api.com/users/v1/10 This is NOT error, just confirmation of success. User is registered.
11 User successfully deleted. https://mmilosevic-diplomski-api.com/users/v1/10 This is NOT error, just confirmation of success. Desired user is deleted.

Errors

Error codes:

Error Code Meaning
/users/v1/[1-6]/td> This error appears when you don't pass mandatory parameters for creating user. In other words, inserting user in database.
/users/v1/[8-9] This error appears when you don't pass mandatory parameters for updating user in database.
sms/v1/[1-4] This error appears when you don't pass mandatory parameters or when passing unallowed/nonexisting parameters when you want to send sms.
message/v1/[1-10] This error appears when you don't pass mandatory parameters or when passing unallowed/nonexisting parameters when you want to insert or update message.
authenticate/v1/[1-3] This error appears when you don't pass mandatory parameters or when passing unallowed/nonexisting parameters when you want to register/log in.
